# MRI Brain Tumor Detection (Deep Learning)

A beginner-friendly deep learning project to classify MRI brain tumor images using a Convolutional Neural Network (CNN).

## What this project does
- Loads MRI images
- Preprocesses the images
- Trains a CNN model using Keras/TensorFlow
- Predicts tumor vs non-tumor images

## Tools and Technologies
- Python
- TensorFlow / Keras
- NumPy
- Pandas
- OpenCV
- Jupyter Notebook
